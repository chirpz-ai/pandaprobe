## Current versions

- **PandaProbe Core OSS service:** `v0.1.5`
- **PandaProbe Python SDK:** `v0.1.4`

## v0.1.5 Core OSS Service

PandaProbe is officially launched with the `v0.1.5` core OSS service release.

This release establishes the first public launch version of the PandaProbe platform, including tracing, evaluation, monitoring, and API-backed workflows for agent observability.

## v0.1.4 Python SDK

The PandaProbe Python SDK is currently at `v0.1.4`.

This SDK release supports the public launch version of PandaProbe and includes the tracing APIs, wrappers, integrations, session tracking, user tracking, scoring, and client utilities used throughout the documentation.
