## What is PandaProbe?

PandaProbe is an open-source agent engineering platform that gives you deep observability into AI agent applications. Use it to trace, evaluate, monitor and debug your AI agents in development and production.

## Why PandaProbe?

PandaProbe is a unified platform for the full agent development lifecycle — from first run to continuous improvement. 

PandaProbe provides:

- **Tracing** for LLM providers, agent development frameworks, and any custom agent
- **Session and user tracking** to follow work across turns and identities
- **Evaluation** to measure quality and regressions
- **Monitoring** for automated evaluation schedules that run periodically
- **Environment-based configuration** so you can tune behavior per deploy without code changes


## Three layers of tracing

1. **Wrappers** — zero-code LLM tracing for major providers
2. **Integrations** — automatic tracing for popular agent frameworks
3. **Manual instrumentation** — full control when you need it
